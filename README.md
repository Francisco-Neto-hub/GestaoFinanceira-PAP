# PAP-Finance
Finance surge como resposta à necessidade de centralizar e otimizar a gestão financeira pessoal. Desenvolvemos uma plataforma digital que permite o registo de movimentos financeiros e a organização por categorias. Utilizando uma metodologia ágil, sistema construído com tecnologias modernas: C# (.NET), React e SQL Server.

> 📖 **Documentação Técnica:** Consulta o nosso [Manual de Integração de Serviços](INTEGRATION.md) para saber como ligar a Interface ao motor do sistema.
